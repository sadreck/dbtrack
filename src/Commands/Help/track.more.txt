NAME
    track   -   Start tracking data.

SYNOPSIS
    dbt track [--track | -t] <tables> [--ignore | -i] <tables>

DESCRIPTION
    This command prepares all the required tables and triggers for the database tracking.

OPTIONS
    <no params>
        If no arguments are passed, all tables will be tracked.

        EXAMPLE

        dbt track

    -t, --track
        If you need to track specific tables, use this parameter.

        EXAMPLE

        dbt track --track table1 table2 table5* some_*_tables

        The command above will track table1, table2, all tables that begin with "table5" and all tables that begin with "some_" and end with "_tables".

    -i, --ignore
        If you need to ignore specific tables, use this parameter.

        EXAMPLE

        dbt track --ignore table7 table5*

        The command above will track ALL tables except table7 and all tables that begin with "table5"
